A Project to simiulate an cluster charging system in order to make predictions concerning potential profits over 10 years when using second life EV batteries for arbitrage

Install all the neccessary libraries.
plotly
dash
pandas
numpy
openpyxl

Next run arbitage_simulation.py
Go to the dash link provided on the terminal to see the visualizations of the simulation.
Open the SimulationData.xlsx to view the raw data from the simulation
